# Personal Portfolio

A modern personal portfolio website built to present my background, experience, projects, and software engineering skills in a clean, interactive format.

## Live Website

🌐 [View Portfolio](https://jadjonaidi.com)

## Overview

This portfolio website is designed as a recruiter-focused personal brand page. It highlights my software engineering background, technical skills, internship experience, and selected projects through a responsive and visually structured layout.

The site emphasizes clarity, storytelling, and usability across both desktop and mobile devices.

---

## Features

- Responsive one-page portfolio layout  
- About, education, experience, projects, and contact sections  
- Horizontal project showcase  
- Smooth navigation and UI interactions  
- Project cards with structured descriptions  
- Mobile-friendly design
